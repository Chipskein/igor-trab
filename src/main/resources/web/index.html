Popular cache

Faça uma requisição via Cache para popular automaticamente o Redis.

Como funciona o cache
  1. A classe LivroCache.listarLivros() verifica se a chave livros:todos existe no Redis.
  2. Se existir (cache hit), desserializa o JSON e retorna imediatamente.
  3. Se não existir (cache miss), consulta o PostgreSQL via JDBC.
  4. O resultado é serializado em JSON e armazenado no Redis com TTL de 60 segundos.
  5. Operações de escrita (criar, atualizar, deletar) invalidam o cache automaticamente.
